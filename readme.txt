This Java program is designed to manage student records using a simple console-based interface. It utilizes a Student class to represent each student, capturing their name, PRN (Permanent Registration Number), and date of birth.

Instructions
Compile the Program:

Ensure you have Java installed on your system.
Open a terminal and navigate to the directory containing the Main.java file.
Compile the program using the command: javac Main.java
Run the Program:

After compilation, run the program with: java Main
Follow the prompts to enter the number of student records and input details for each student.
View Records:

The program will display the entered student records with details such as name, PRN, and date of birth.
File Structure
Main.java: Contains the main method and program logic for input and display of student records.
InputUser.java: Class responsible for handling user input and returning an array with student details.
Student.java: Represents the Student entity with attributes such as name, PRN, and date of birth.
Usage Tips
Follow the on-screen prompts to input the number of student records and details for each student.
The program will then display the entered records in a structured format.
